- **📅 21 May (Tue):**  
  🔄 Applied control flow concepts such as loops (`for`, `while`) and conditional statements (`if-else`, `switch`) to solve logical problems and control program behavior effectively.

- **📅 22 May (Wed):**  
  📊 Worked extensively with arrays and objects to manipulate data collections.  
  🚀 Used advanced array methods like `map()`, `filter()`, and `reduce()` to process and transform data efficiently.

- **📅 23 May (Thu):**  
  🌐 Learned DOM (Document Object Model) manipulation techniques to dynamically select, modify, and update HTML elements using JavaScript.  
  🎯 Implemented event handling for user interactions such as clicks, inputs, and form submissions.

- **📅 24 May (Fri):**  
  📝 Developed a fully functional, responsive To-Do List application from scratch.  
  🔧 Implemented features to add, edit, and delete tasks dynamically, incorporating DOM manipulation and event listeners.

- **📅 25 May (Sat):**  
  🔍 Conducted thorough code review and optimization of the To-Do List app.  
  🛠 Improved code readability, performance, and ensured responsiveness across devices.

### 🔹 Week 3: Backend Development

- **📅 27 May (Mon):**
  👥First, conducted a meeting with the HR team to discuss the programming languages to be used, approaches for problem-solving, the tasks to be performed during the internship, and 
    details regarding the mini project.
  
  ⚙️ Installed and configured Node.js runtime environment, explored npm (Node Package Manager) for managing dependencies.  
  🚀 Set up the first basic backend server, understanding the event-driven architecture of Node.js.

- **📅 28 May (Tue):**  
  🛣 Built RESTful APIs using Express.js framework.  
  🔧 Implemented route handling, middleware functions, and learned about request-response lifecycle.

  💾 Studied MongoDB basics including collections, documents, and queries.

- **📅 31 May (Fri):**  
  📝 Developed a complete Blog API covering all CRUD operations using Node.js, Express, and MongoDB stack.  
  🛠 Focused on building scalable and maintainable backend services.
